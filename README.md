# PAM Consulting — Корпоративный сайт

Многостраничный адаптивный веб-сайт для консалтинговой компании. Проект демонстрирует навыки верстки сложной структуры и интеграции сторонних сервисов.

## 🔗 Ссылки
- Live Demo: https://pam-gules.vercel.ap
- GitHub: [https://github.com/kotymurk/PAM](https://github.com/kotymurk/PAM)

## 🛠 Технологии
- HTML5 & CSS3 (Семантическая верстка, Flexbox, Grid)
- JavaScript (ES6+) — логика меню, аккордеонов и слайдеров
- BEM Methodology — для чистого и масштабируемого кода
- Swiper.js — для реализации интерактивных слайдеров
- Formspree API — для обработки и отправки данных из форм на Email

## 🚀 Особенности проекта
- Полная адаптивность: Сайт корректно отображается на десктопах, планшетах и мобильных устройствах.
- Интерактивные элементы: Реализованы выпадающие меню, FAQ-аккордеоны и карусели изображений.
- Рабочая форма обратной связи: Все заявки приходят на почту владельцу сайта.
- Кроссбраузерность: Поддержка всех современных браузеров.

## 📂 Структура проекта
- /js — скрипты для интерактивных элементов
- /css — стили (разбиты по методологии БЭМ)
- /img — оптимизированные графические ресурсы
